# ИИ-игрок лиги — GridBot

Раз в неделю (по четвергам, до дедлайна) ставит прогноз топ-10 наравне с людьми: смотрит пул гонки, свои прошлые результаты сезона и итоги последних 3 сезонов F1 (через Jolpica), просит Gemini предположить порядок. Если Gemini недоступна или ответ невалиден — подставляется прогноз по текущему месту в чемпионате (без падения всего прогона). По четвергам после сохранения прогноза публикует в общий Telegram-чат короткое объяснение выбора.

Играет как обычный участник лиги (аккаунт `GridBot`, миграция `0014_gridbot_user.sql`) — те же очки, та же видимость в списке проголосовавших, тот же общий зачёт. Никакого отдельного UI.

## Разовая настройка

1. Открыть [Google AI Studio](https://aistudio.google.com/) → **Get API key** → создать новый ключ (бесплатный тариф — лимитов с запасом хватает при частоте «раз в неделю»).
2. В корневом `.env` (НЕ `.env.example`) добавить:
   ```
   GEMINI_API_KEY=<ключ из шага 1>
   ```
3. В GitHub-репозитории: **Settings → Secrets and variables → Actions → New repository secret** — добавить `GEMINI_API_KEY` с тем же значением (нужен для автозапуска по расписанию).

## Ручной запуск / проверка

```bash
cd scripts/ai-player
npm install   # один раз
npm run predict
```

Сработает только если есть открытая гонка с дедлайном на этой неделе — иначе выведет и завершится без ошибки: `aiplayer: нет открытой гонки с дедлайном на этой неделе, пропускаем`.

При успехе выводит: `aiplayer: прогноз на "<Гонка>" сохранён — VER-NOR-...` и (если Gemini ответила успешно — независимо от дня недели, скрипт день не проверяет) `aiplayer: объяснение отправлено в Telegram`. Четверг — это только расписание крона в GitHub Actions (mode `aiplayer` запускается по нему только по четвергам утром); при ручном запуске `npm run predict` в любой день, если гонка открыта и ключ Gemini рабочий, сообщение реально уйдёт в общий Telegram-чат.

Проверить результат: открыть вкладку «Прогноз» под своим аккаунтом на этой гонке — в списке «Проголосовали» должен появиться GridBot (сам прогноз, как и у всех, скрыт до дедлайна).
